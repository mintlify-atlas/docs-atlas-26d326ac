## What is RAGler?

RAGler is an open-source **RAG knowledge operations platform** that gives you full control over your retrieval-augmented generation pipeline. Unlike black-box solutions, RAGler puts humans in the loop — every chunk of knowledge is reviewed, edited, and approved before it reaches your vector database.

## How it works

RAGler follows a structured pipeline that keeps you in control at every step:

<Steps>
  <Step title="Ingest">
    Pull content from web URLs, uploaded files (PDF, DOCX, TXT, MD), or paste text manually. Each ingest creates a **draft session** in Redis.
  </Step>
  <Step title="Review & Edit">
    Inspect AI-generated chunks in your session. Split, merge, rewrite, or add chunks manually. Every change is saved in the draft until you're ready.
  </Step>
  <Step title="Preview">
    Lock the session and validate all chunks before committing. Preview shows exactly what will be published.
  </Step>
  <Step title="Publish">
    Atomically write validated chunks to a **Qdrant vector collection**. Old chunks are replaced, embeddings are generated, and your knowledge base is live.
  </Step>
</Steps>

## Architecture at a glance

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   Frontend   │────▶│   Backend    │────▶│    Qdrant    │
│  (Next.js)   │     │  (NestJS)    │     │  (vectors)   │
└──────────────┘     └──────┬───────┘     └──────────────┘
                            │
                     ┌──────┴───────┐
                     │    Redis     │
                     │  (sessions)  │
                     └──────────────┘
```

| Component | Technology | Role |
|-----------|-----------|------|
| Frontend | Next.js 16, Tailwind CSS 4 | User interface for all operations |
| Backend API | NestJS 11 | REST API with Swagger, built-in MCP server |
| Redis | Redis Alpine | Draft session storage with TTL |
| Qdrant | Qdrant | Published chunk vectors + metadata |
| OpenAI | OpenAI API | LLM chunking, embeddings, agent chat |

## Prerequisites

Before installing RAGler, ensure you have:

- **Node.js** 20+
- **pnpm** 9+
- **Docker** (latest)
- **OpenAI API key**

<Note>
  RAGler uses Docker to run Redis and Qdrant. You don't need to install these separately — `docker compose up -d redis qdrant` handles everything.
</Note>
